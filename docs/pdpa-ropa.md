# บันทึกรายการกิจกรรมการประมวลผลข้อมูลส่วนบุคคล (RoPA)

> Record of Processing Activities — ตาม พ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 **มาตรา 39**
> เอกสารภายใน (ไม่เผยแพร่) — ปรับปรุงเมื่อมีการเปลี่ยนแปลงกิจกรรมการประมวลผล และเก็บไว้แสดงต่อ สคส. (PDPC) เมื่อร้องขอ

**หมายเหตุ:** เอกสารนี้เป็นแบบร่างเพื่อเริ่มต้น ควรให้ที่ปรึกษากฎหมาย/DPO ทบทวนและปรับตามการใช้งานจริง

## 3. มาตรการรักษาความมั่นคงปลอดภัย (สรุป)

- เข้ารหัสระหว่างส่ง (HTTPS/TLS) ทั้งระบบ
- ควบคุมสิทธิ์เข้าถึงแบบแยกองค์กรด้วย Row-Level Security (RLS) — ผู้ใช้เห็นเฉพาะข้อมูลหอตนเอง
- ไฟล์เอกสาร (บัตร ปชช./สัญญา) และสลิปโอนเงิน จำกัดการเข้าถึงเฉพาะองค์กรเจ้าของข้อมูล (Storage RLS)
- แยกสิทธิ์แอดมินแพลตฟอร์มเป็นราย section (least privilege)
- Rate limiting กันการยิงถี่/brute-force
- บันทึกกิจกรรม (audit log) การดำเนินการสำคัญ
- Secret/service key เก็บฝั่ง server เท่านั้น ไม่หลุดสู่ client
